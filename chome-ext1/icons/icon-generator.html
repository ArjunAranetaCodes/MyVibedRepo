Chrome Extension Icon Generator

This page shows the icon sizes needed for your Chrome extension. To create actual PNG files:

  1. Right-click on each icon below
  2. Select "Save image as..." or "Copy image"
  3. Save as PNG files with the correct names: icon16.png, icon32.png, icon48.png, icon128.png
  4. Place them in the icons/ folder

Note: These are placeholder icons. For production, create proper PNG images with your extension's branding.
